## Этапы настройки
1. **Linux и Bash**: настройка SSH, UFW, написание скрипта `cleanup.sh`.
2. **Git и GitHub**: инициализация репозитория, связь с удалённым репозиторием.
3. **Docker**: контейнеризация Hugo, первая сборка сайта.
4. **CI/CD**: автоматический деплой через GitHub Actions при пуше в main.
5. **Веб-сервер и HTTPS**: настройка Caddy, получение SSL-сертификата для devops.ai-donate.ru.
6. **Мониторинг**: Prometheus + Grafana, Node Exporter на всех серверах, алерт на память.
7. **База знаний**: создание раздела на сайте для документирования инцидентов.

## Ключевые решённые проблемы
- **Восстановление SSH на сервере 3**: полная переустановка openssh-server, ручное создание конфига, обход ошибки AF_VSOCKID.
- **Подмена символов в VNC**: обнаружена при попытке настроить парольный вход (символ `_` заменялся на `-`).
- **Старые job'ы в Grafana**: очистка хранилища Prometheus (`rm -rf /var/lib/prometheus/*`).
- **Ограничения памяти на сервере с ботами**: мониторинг и алерт при доступной памяти < 15%.
