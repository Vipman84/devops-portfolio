CI/CD - Continuous Integration и Continuous Deployment

CI/CD — это методология, которая включает в себя автоматизированную тестирование и проверку代码, а также автоматизированный процесс deployment(code push) на production-Environment.

Четыре main components CI/CD:

  • CI (Continuous Integration): Automated tests and code reviews for every code change
  • CD (Continuous Deployment): Automated process of deploying changes to production environment
  • CD/CI: Combines both concepts, automating both testing and deployment processes

Преимущества CI/CD:

Преимущества CI/CD

  • Быстрый процесс deployments, уменьшение времени на rollout;
  • Улучшение качества кода, снижение количество ошибок;
  • Восприятие более быстрого результата с помощью Continuous Integration и Continuous Deployment.
