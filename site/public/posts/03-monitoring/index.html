Неделя 3: Мониторинг и алерты

Prometheus + Grafana

  • Установлен Prometheus на основном сервере.
  • Node Exporter собирает метрики с обоих VPS.
  • Grafana визуализирует данные, дашборд Node Exporter Full.
  • Настроен алерт: если на целевом сервере остаётся меньше 15% памяти.

Доступ

  • Grafana открывается через SSH-туннель или локально.
  • Включён анонимный просмотр для удобства.

Конфигурация

  • Все конфиги хранятся в репозитории в папке monitoring/.
