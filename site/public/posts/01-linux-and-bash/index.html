Неделя 1: Linux, Bash и первый сервер

Цель

Освоить базовую настройку Linux-сервера, обеспечить безопасный доступ и написать первый скрипт автоматизации.

Что сделано

  • Арендован VDS с Debian 12, настроен нестандартный порт SSH (19070).
  • Создан пользователь devops, запрещён прямой вход для root.
  • Настроен фаервол UFW (разрешён только нужный порт).
  • Настроен вход только по SSH-ключу, парольная аутентификация отключена.
  • Написан Bash-скрипт cleanup.sh для ротации логов с архивированием старых записей.
  • Инициализирован Git-репозиторий, код отправлен на GitHub.

Итог

Сервер защищён и готов к дальнейшей работе. Заложены основы автоматизации и контроля версий. Следующий шаг — контейнеризация и CI/CD.
