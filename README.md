# DevOps Учебник

Учебный проект по DevOps, в котором шаг за шагом разворачивается сайт-учебник на собственном сервере.

- 🌐 **Сайт:** [https://devops.ai-donate.ru](https://devops.ai-donate.ru)
- 🛠 **Репозиторий:** [Vipman84/devops-portfolio](https://github.com/Vipman84/devops-portfolio)

## Что уже сделано
- Настроен сервер Debian 12, SSH по ключу, фаервол UFW
- Написан Bash-скрипт для ротации логов с архивированием
- Настроен Git и GitHub
- Собран статический сайт на Hugo
- Развёрнут веб-сервер на отдельном VPS с обратным прокси (Caddy) и автоматическим HTTPS
- Настроен автоматический деплой через GitHub Actions (CI/CD)
- Подключён мониторинг (в процессе)

## Структура проекта
- `scripts/` — служебные скрипты (очистка логов и др.)
- `site/` — исходники сайта (Hugo)
- `deploy.sh` — скрипт деплоя на продакшен-сервер
- `.github/workflows/` — CI/CD пайплайн
